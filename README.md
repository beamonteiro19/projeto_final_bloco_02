# Projeto Final – Bloco 02 Farmácia API

API desenvolvida em **NestJS** como projeto final do Bloco 02, simulando o sistema de uma **farmácia**, com foco em organização de dados, regras de negócio e boas práticas no desenvolvimento backend.

---

## Contexto do Projeto

Este projeto tem como objetivo aplicar conceitos fundamentais de:
- Arquitetura em camadas
- Desenvolvimento de APIs REST
- Boas práticas com NestJS
- Testes automatizados

O sistema representa operações comuns de uma farmácia, como o gerenciamento de produtos, categorias e regras relacionadas.

---

##  Funcionalidades

- Cadastro de produtos farmacêuticos
- Gerenciamento de categorias
- Atualização de informações
- Remoção de registros
- Consulta de dados
- CRUD completo
- Estrutura modular com:
  - Controllers
  - Services
  - Entities
- Testes unitários e testes de integração (e2e)

---

##  Tecnologias Utilizadas

- **Node.js**
- **NestJS**
- **TypeScript**
- **Jest** (testes)
- **npm**

---


## Como rodar o projeto

npm install
npm run start:dev
